5月30日（土）・6月7日（土）公開向け・3カテゴリ別候補

作業系 — この時期の庭仕事・管理テーマ
★★★
梅雨前に絶対やっておくべき庭仕事5選【植木屋が教える】

除草・排水改善・風通し確保のための軽剪定・病害虫の早期確認・鉢の蒸れ対策を、プロ視点でリスト化。「やることが多くて迷う」初心者に刺さるチェックリスト型。

根拠：梅雨前の庭仕事は毎年定番テーマ。5/30公開なら梅雨入り直前でタイムリー 推奨：5月30日（土）
動画構成メモ（目安8〜12分）
  1. オープニング（30秒）：「梅雨で庭が大変になる前に今すぐやることがある」フック
  2. なぜ梅雨前が重要か（1〜2分）：蒸れ・病害虫の被害パターンをビジュアルで
  3. 作業5選を順番に解説（5〜7分）：除草→排水確認→軽剪定→病害虫確認→鉢の置き場
  4. まとめ＋クロージング（1分）：「これだけやれば梅雨を乗り切れる」
★★☆
ツツジ・サツキの花後剪定 今月中にやらないと来年咲かない！

花後剪定の「タイムリミット（6月中旬）」を強調し、方法・道具・切り戻し量を分かりやすく解説。緊急性タイトルで既存の庭木オーナーに直接刺さる。

根拠：「今月中にやらないと」系タイトルは毎年6月に安定再生 推奨：6月7日（土）
動画構成メモ（目安6〜10分）
  1. オープニング（30秒）：「これを6月中旬までにやらないと来年花が咲かない」
  2. なぜ花後剪定が重要か（1分）：花芽形成の仕組みを簡単に説明
  3. 剪定の方法（4〜6分）：ツツジ・サツキそれぞれの切り方・道具・注意点を実演
  4. よくある失敗パターン（1〜2分）：切りすぎ・遅すぎの例
  5. まとめ（30秒）：「今週中にやれば来年も咲く」
お庭におすすめの植物 — 低木・シンボルツリーなど
★★★
手入れ不要で増やせる！梅雨に映えるアジサイ品種3選【初心者向け】
低木

アナベルなど初心者でも育てやすい品種を3つ絞り込み、梅雨の挿し木タイミングと合わせて紹介。「手入れ不要」「増やせる」の二重訴求でターゲット3層すべてに刺さる。

根拠：アジサイは梅雨の象徴植物・新品種人気上昇中・挿し木タイミングとも重なる三拍子 推奨：5月30日（土）
動画構成メモ（目安8〜12分）
  1. オープニング（30秒）：「梅雨に映える＋手入れ不要＋増やせる品種がある」フック
  2. 品種①：アナベル（白花・強健・剪定しやすい・地植えOK）
  3. 品種②：ダンスパーティー等 / 新品種（コンパクト・鉢OK）
  4. 品種③：カシワバアジサイ（大型・管理少・和洋兼用）
  5. 挿し木のコツ（梅雨が最適な理由・失敗しない方法）（2〜3分）
  6. まとめ（30秒）：「この3種なら初心者でも失敗しない」
★★☆
シンボルツリー選びで失敗しない！アオダモをおすすめする理由【プロが解説】
高木

アオダモを軸に、デメリット（落葉・成長速度）も包み隠さず解説するプロ視点の誠実系動画。「失敗しない」タイトルで検索流入も狙える。

根拠：春〜6月は植え付け最適期でシンボルツリー検討層が最も活発。デメリット解説の「正直動画」は信頼と滞在率が高い 推奨：6月7日（土）
動画構成メモ（目安10〜15分）
  1. オープニング（30秒）：「シンボルツリー、植える前にこれを見てください」
  2. アオダモの特徴（メリット）（3〜4分）：樹姿・落葉・成長速度・和洋どちらにも合う
  3. デメリットも正直に（2分）：落葉の掃除・成長すると剪定必要・価格帯
  4. どんな家に向いているか（2分）：向き・土・場所の条件
  5. 植え方・水やりのコツ（2〜3分）
  6. まとめ（1分）：「これが分かれば後悔しない選び方ができる」
