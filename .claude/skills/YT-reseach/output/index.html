企画アイデア

5月30日（土）・6月6日（土）公開向け・3カテゴリ別候補

作業系 — この時期の庭仕事・管理テーマ ★評価基準：競合実績×タイムリー性×チャンネル相性の3軸
★★★
梅雨前に絶対やっておくべき庭仕事5選【植木屋が教える】

排水確認・密集枝の間引き・病害虫の早期チェック・鉢の蒸れ対策・雑草の一掃を、プロ視点でリスト化。チェックリスト型で「何から手をつけるか分からない」初心者に直接刺さる。

根拠：梅雨前の庭仕事は毎年定番テーマ。5/30公開なら梅雨入り直前でタイムリー 推奨：5月30日（土）
動画構成メモ（目安8〜12分）
  1. オープニング（30秒）：「梅雨で庭が荒れる前に今すぐやることがある」
  2. なぜ梅雨前が重要か（1〜2分）：蒸れ・病害虫のビジュアル説明
  3. 作業5選を順番に解説（5〜7分）：①除草（根から抜く） ②排水確認（水はけチェック） ③密集枝の間引き剪定 ④葉裏の病害虫確認 ⑤鉢の置き場・蒸れ対策
  4. まとめ（1分）：「これだけやれば梅雨を乗り切れる」
サムネイルアイデア

「梅雨前にやること5選」テキスト大文字 ＋ 庭の作業イメージ写真 ＋ 「今すぐ！」「植木屋が教える」フック文字。背景は緑で季節感を出す。

★★☆
アジサイの花後剪定 今月中にやらないと来年咲かない！

「6月下旬がタイムリミット」という緊急性を前面に出し、剪定の方法・切る位置・道具を分かりやすく実演する。既に庭にアジサイがある層に直接刺さる。

根拠：毎年6月に安定再生。カーメン君等のベンチマークも毎年このテーマを投稿 推奨：6月6日（土）
動画構成メモ（目安6〜10分）
  1. オープニング（30秒）：「これを6月下旬までにやらないと来年花が咲かない」
  2. 花芽形成の仕組み（1分）：なぜ今剪定が重要かを簡単に
  3. 剪定方法の実演（4〜6分）：切る位置（花の2〜3節下で切る）・道具・注意点（アジサイ種類別の違いも触れる）
  4. よくある失敗（1〜2分）：切りすぎ・遅すぎの例（来年咲かなかった失敗談を自分の言葉で）
  5. まとめ（30秒）：「今週中にやれば来年も咲く」
サムネイルアイデア

アジサイのアップ写真 ＋「今月中！」大文字 ＋「やらないと来年咲かない」赤背景テキスト。緊急感を色と文字サイズで表現。

お庭におすすめの植物 — 低木・シンボルツリーなど
★★★
手入れ不要で増やせる！梅雨に映えるアジサイ品種3選【初心者向け】
低木

アナベル・ノリウツギ・カシワバアジサイを軸に、それぞれの特徴・管理しやすさ・挿し木タイミングを紹介。「手入れ不要」「増やせる」の二重訴求でターゲット3層すべてに刺さる。

根拠：アジサイは梅雨の象徴・新品種人気上昇中・挿し木タイミングとも重なる三拍子 推奨：5月30日（土）
動画構成メモ（目安8〜12分）
  1. オープニング（30秒）：「梅雨に映える＋手入れ不要＋増やせる品種がある」フック
  2. 品種①：アナベル（白花・強健・剪定しやすい・地植えOK・大株になる）（2〜3分）
  3. 品種②：ノリウツギ（秋まで咲き続ける・花が変わる・寒さに強い）（2〜3分）
  4. 品種③：カシワバアジサイ（大型・紅葉も美しい・和洋兼用・花期が長い）（2〜3分）
  5. 梅雨の挿し木チャンス（1分）：「今が増やせる時期」ボーナス情報で視聴者の行動を引き出す
  6. まとめ（30秒）

アナベル・ノリウツギ・カシワバの3枚並べ ＋「梅雨に映える」「手入れ不要」テキスト。雨上がりの瑞々しいカット使用が理想。

★★☆
シンボルツリー選びで後悔しない！プロが本音で教えるおすすめ3選
高木

オリーブ・ヤマボウシ・アオダモを軸に、デメリットも包み隠さず解説する「正直動画」。「プロが本音で」タイトルと「後悔」ワードで検索流入と視聴完了率を両取りする。

根拠：春〜6月はシンボルツリー検討層が最も活発。「後悔」「本音」タイトルが過去の人気動画に共通するパターン 推奨：6月6日（土）
動画構成メモ（目安10〜15分）
  1. オープニング（30秒）：「これを選んで後悔した、というシンボルツリーを正直に話します」
  2. 樹種①：オリーブ（洋風で人気だが関東以南では成長が旺盛で大きくなりすぎる・剪定を怠ると暴れる）（3〜4分）
  3. 樹種②：ヤマボウシ（落葉で冬に丸裸になる点を知らずに選ぶ後悔が多い・成長速度も予想外）（3分）
  4. 樹種③：アオダモ（細幹が人気だが成長後に存在感が変わる・日当たり要件の誤解）（3分）
  5. 選び方まとめ（1〜2分）：「家の雰囲気・管理の手間・好みで決める基準」＋後悔しにくい樹種を1〜2個紹介して締める

イベント・特集系 — バズパターンを活用した特集・知識共有
★★★
