# 図解タイプ ガイド

## タイプ判定フロー

```
依頼内容を確認
  ├─ 「トレンド」「流行」「今何が人気か」
  │   → タイプ1: トレンド分析図解
  │
  ├─ 「ベンチマーク」「競合」「あのYouTuber」「最近の動向」
  │   → タイプ2: ベンチマーク分析図解
  │
  ├─ 「キーワード」「検索」「いつ投稿すべきか」「季節性」
  │   → タイプ3: キーワード分析図解
  │
  └─ 複数の情報を統合したい / 台本ネタを探している
      → タイプ4: 総合企画立案図解
```

---

## タイプ1: トレンド分析図解

**目的**: 今ガーデニング業界で何が盛り上がっているかを一覧で把握する

### 必須セクション

1. **トレンドサマリー**（インサイトボックス）
   - 今月の注目トレンドを1〜2文で
   
2. **トレンド一覧テーブル**
   - トレンド名 / 動向（↑→↓）/ ピーク時期 / 企画メモ
   - 5〜10件程度

3. **注目トレンド 詳細**（トップ3を深掘り）
   - なぜ今盛り上がっているか
   - どんな視聴者層に刺さるか
   - 類似コンテンツとの差別化ポイント

4. **企画アイデアセクション**（必須・最後に配置）
   - トレンドから導いた企画3〜5案

### リサーチ指示（Phase 0 補足）

```
調査対象：
- 園芸・ガーデニング関連のSNSトレンド（X, Instagram）
- 季節の植物・作業（今月〜来月に旬のもの）
- ガーデニング商品・サービスの話題（新製品、流行アイテム）
- イベント・シーズンとの連動（春植え、冬支度など）
```

## タイプ2: ベンチマーク分析図解

**目的**: ベンチマークYouTuberの最新動向を把握して、企画の方向性を参考にする

### 必須セクション

1. **チャンネル概要カード**
   - 登録者数・投稿頻度・最近の傾向

2. **直近動画 分析テーブル**
   - タイトル / 推定再生数 / テーマ / コメント欄の反応傾向
   - 5〜10本

3. **伸びている動画の共通パターン**
   - テーマ・構成・タイトルのパターン分析
   - 「なぜ伸びているか」の仮説

4. **最近の方向性サマリー**
   - ここ3ヶ月でチャンネルが変わったこと
   - 自チャンネルへの示唆

5. **企画アイデアセクション**（必須・最後に配置）
   - ベンチマークの動向から導いた企画3〜5案

### リサーチ指示（Phase 0 補足）

```
調査対象：
- 指定YouTuberの直近10本の動画タイトル
- 各動画の推定再生数（多い/普通/少ない の3段階でも可）
- サムネイルや投稿タイミングの傾向
- コメント欄で多い反応・質問
- 最近始めた/やめたコンテンツ傾向
```

---

## タイプ3: キーワード分析図解

**目的**: 特定のキーワードがいつ・どれくらい検索されるかを把握して、投稿タイミングを最適化する

### 必須セクション

1. **キーワードサマリー**
   - 対象キーワードの概要・競合度・特徴

2. **季節性グリッド（12ヶ月）**
   - 各月の検索量レベル（高/中/低）
   - ピーク月を強調表示

3. **関連キーワード一覧**
   - バッジ付き（注目 / 上昇中 / 季節性 / ねらい目）
   - 優先度順に並べる

4. **競合コンテンツ傾向**
   - どんな動画が上位にいるか
   - 差別化できる切り口はあるか

5. **投稿タイミング推奨**
   - いつ投稿するのが最適か
   - 理由を添える

6. **企画アイデアセクション**（必須・最後に配置）
   - キーワードを活用した企画3〜5案

### リサーチ指示（Phase 0 補足）

```
調査対象：
- 対象キーワードの検索量の季節パターン
- 「[キーワード] やり方」「[キーワード] 初心者」など複合キーワード
- YouTube検索結果の上位動画の傾向（タイトル・再生数・投稿日）
- Google トレンドのデータ（例年の動き）
```

## タイプ4: 総合企画立案図解

**目的**: トレンド・ベンチマーク・キーワードの情報を統合して、今月・来月の企画候補を意思決定する

**使いどき**: 毎週の台本作成前の情報整理、四半期ごとのチャンネル戦略見直し

### 必須セクション

1. **今月の状況サマリー**（インサイトボックス）
   - トレンド・競合・キーワードを3点で要約

2. **トレンドスナップショット**（タイプ1の凝縮版）
   - 上昇中のトレンド 3〜5件をカード形式で

3. **競合の動き**（タイプ2の凝縮版）
   - ベンチマークの最近の方向性を一言で

4. **注目キーワード**（タイプ3の凝縮版）
   - 今月〜来月に旬のキーワードを一覧

5. **企画候補マトリクス**
   - 横軸：独自性（低〜高）、縦軸：検索需要（低〜高）
   - 候補企画を配置（テキストベースのマトリクスでもOK）

6. **企画アイデアセクション**（必須・最後に配置）
   - 優先度順に5案以上

### リサーチ指示（Phase 0 補足）

```
以下の3軸で調査してください：

1. トレンド軸
   - 今月〜来月のガーデニングシーズンイベント
   - SNS上の話題（季節の植物・トラブル・お役立ち情報）

2. 競合軸
   - ベンチマークチャンネル1〜2社の直近5本の動画

3. キーワード軸
   - 今月〜来月にピークを迎える検索キーワード 5〜10件
```

---

## どのタイプにも共通するルール

1. **企画アイデアセクションは必ず最後** — 図解の締めは常にアクション（企画案）
2. **根拠を省かない** — 「なぜ今か」が書かれていない企画案は不完全
3. **5項目以上の一覧はテーブルかカードグリッドで** — テキストの箇条書きにしない
4. **最重要情報はインサイトボックスで強調** — 読者（＝自分）がまず目に入る場所に
